# Personal Diary Management System (C)

A menu-driven **Personal Diary Management System** implemented in **C**, designed to demonstrate the practical use of **Data Structures and Algorithms**.

This project allows users to securely maintain diary entries with features such as add, update, delete, search, sort, and undo operations. It is built as part of my DSA learning journey and focuses on correctness, clarity, and real-world applicability.
